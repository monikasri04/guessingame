Guessing Game Project: Step-by-Step Guide

Step 1: Create Project Folder

  1. Open a terminal.
  2. Create a folder for your project: 
    mkdir guessinggame-project
cd guessinggame-project

Step 2: Create guessinggame.sh

  1. Create the Bash script file: 
    touch guessinggame.sh
chmod +x guessinggame.sh
  2. Open it in an editor and paste the following code: 
    #!/bin/bash

count_files() {
  echo $(ls -1 | wc -l)
}

prompt_guess() {
  echo -n "Guess how many files are in the current directory: "
  read guess
}

num_files=$(count_files)

while true; do
  prompt_guess
  if [[ $guess -lt $num_files ]]; then
    echo "Too low! Try again."
  elif [[ $guess -gt $num_files ]]; then
    echo "Too high! Try again."
  else
    echo "Congratulations! You guessed correctly."
    break
  fi
done

Step 3: Create the Makefile

  1. Create a file called Makefile: 
    touch Makefile
  2. Paste this content: 
    # Variables
SCRIPT=guessinggame.sh
README=README.md
DATE=$(shell date)

all: $(README)

$(README):
	@echo "# Guessing Game Project" > $(README)
	@echo "" >> $(README)
	@echo "Generated on: $(DATE)" >> $(README)
	@echo "" >> $(README)
	@echo "Number of lines in $(SCRIPT):" >> $(README)
	@wc -l < $(SCRIPT) >> $(README)

clean:
	rm -f $(README)

Step 4: Test the Bash Script

  1. Run the game: 
    bash guessinggame.sh
  2. The program will repeatedly ask for guesses until you enter the correct number of files.

Step 5: Generate README.md

  1. Run Makefile: 
    make
  2. Check README.md content: 
    cat README.md

Step 6: Initialize Git and Push to GitHub

  1. Initialize Git: 
    git init
  2. Add files and commit: 
    git add guessinggame.sh Makefile README.md
git commit -m "Initial commit"
  3. Create a repository on GitHub and push: 
    git remote add origin <your-github-url>
git branch -M main
git push -u origin main

Step 7: Set up GitHub Pages

  1. Go to your GitHub repository → Settings → Pages.
  2. Select main branch and root folder as source.
  3. Save. GitHub will provide a URL for your Pages site.
